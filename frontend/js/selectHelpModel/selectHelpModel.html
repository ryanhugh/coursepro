Selecting Prerequisites

Dark Blue classes are required prerequisites. These appear if a class requires that you take all of the classes from a list of classes. For example, if class A requires that you take classes B, C, and D, all three prerquisites will show up as dark blue.
With Light Blue classes you have a choice. You can pick any of the light blue prerequisites connected to a class to satisfiy that class. For example, if class A requires class B, C, or D. Classes B, C, and D will show as light blue underneath class A. When one of them is selected, the others will disappear and the remaining one will turn dark blue.
